# Context_Aware_Chatbot_With_LangChain_JS

## This project is built on Vanilla JS

## LangChain is a framework that helps developers build context-aware reasoning applications. It helps in factoring out some common abstractions.

## LangChain JS helps us build AI web applications.

# What is in this project?
1. Embeddings
    * The language that AI understands. Embedding is a mathematical      concept that refers to placing one object into a different space. Embeddings are plotted in vectors to show relationships between words.

    * For storing our embeddings for this application, we will store them in Supabase, which is a popular and user friendly vector store.

2. Working with vector states
3. Building Templates
4. Creating prompts from templates
5. Setting up chains
6. The .pipe() method
7. Retrieving from vector states
8. The RunnableSequence class
